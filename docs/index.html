Frequently Asked Questions

Is this extension safe to use?

Yes, this extension is completely safe. It only accesses LinkedIn pages you visit and stores data locally in your browser. It does not transmit any personal information to external servers except when using the AI evaluation feature (which requires your own OpenAI API key) or Virk.dk integration.

Do I need an OpenAI API key?

No, the AI evaluation feature is optional. You can use all other features without an API key. The AI evaluation helps score leads based on your custom criteria, but basic lead extraction works without it.

What permissions does the extension need?

The extension requests access to LinkedIn domains and browser storage. It needs these permissions to:

  • Extract data from LinkedIn pages
  • Store leads locally in your browser
  • Download CSV/JSON files
  • Access the OpenAI API (optional)
  • Access Virk.dk for Danish company data (optional)
Can I use this for commercial purposes?

Yes, you can use the extension for commercial lead generation and business development. The extension is designed for professional use and includes features specifically for sales and marketing teams.

How many leads can I extract?

The extension can handle hundreds of leads efficiently. The "Deep Scan ALL" feature can process up to 100 pages of LinkedIn search results automatically. All data is stored locally in your browser with automatic deduplication.

Does it work on mobile devices?

No, this is a Chrome extension that only works on desktop Chrome browsers. LinkedIn's mobile interface is different and doesn't support the same automation features.

What if LinkedIn changes their interface?

The extension uses robust selectors and includes error handling. If LinkedIn makes significant changes, the extension will be updated. You can check the GitHub repository for the latest version and updates.

Can I export data to my CRM?

The extension exports data as CSV or JSON files, which can be imported into most CRM systems. The data includes all extracted fields plus AI scores and enrichment data, making it ready for CRM integration.

Troubleshooting

Extension not appearing after installation

Solution: Make sure you've enabled "Developer mode" in Chrome extensions and clicked "Load unpacked" with the correct folder. The extension icon should appear in your toolbar after pinning it.

Buttons not showing on LinkedIn

Solution: Refresh the LinkedIn page and make sure you're on a search results page. The extension only activates on LinkedIn search result pages.

AI evaluation not working

Solution: Check that you've entered a valid OpenAI API key in the extension settings. Make sure your API key has sufficient credits and the correct permissions.

Deep scan getting stuck

Solution: LinkedIn may have rate limiting. Try increasing the delay between scans in the settings, or run scans during off-peak hours. Also check your internet connection.

Data export failing

Solution: Make sure your browser allows downloads. Check that you have sufficient disk space and that no security software is blocking the download.

Virk.dk integration not working

Solution: This feature only works for Danish companies. Make sure the company has a valid CVR number and that the Virk.dk service is available.

Extension running slowly

Solution: Clear your browser cache and restart Chrome. If using Deep Scan ALL, reduce the number of parallel tabs or increase delays between operations.

Connect automation not working

Solution: LinkedIn may have changed their interface. Check the extension settings and try adjusting the message templates. Some accounts may have sending limits.
