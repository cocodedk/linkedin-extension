## 8. Success Criteria

* A user can:

  * Run a LinkedIn search → click *Scan Results* → see leads appear in the table.
  * Export them to CSV/JSON with FITS evaluation columns.
  * Optionally evaluate with OpenAI and see progress + summaries inline.
  * Repeat the flow in Chrome or Firefox with matching behavior.
* All within <5 clicks, <5 minutes.
